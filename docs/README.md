# Documentation index

This folder contains the public-facing narrative docs and dataset governance notes. If you’re new, start with `START_HERE.md`, then read `REPORT.md`.

- `START_HERE.md` — 2–10 minute orientation and reading path
- `REPORT.md` — paper-style narrative report (methods + results + limitations)
- `EXPLAINER.md` — plain-language guide to metrics (Brier, AUROC, thresholds, subgroup gaps)
- `PUBLISHING.md` — what can/can’t be published + how to generate `public_export/`
- `dataset_card_nij.md` — NIJ dataset provenance, labels, and prohibited uses
- `dataset_card_compas.md` — COMPAS dataset provenance, labels, and prohibited uses
- `datasets/` — notes on integrating additional datasets (ICPSR/NACJD, NCRP, etc.)
- `model_card_template.md` — template for a model card describing training/evaluation/limitations
