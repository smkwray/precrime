# Additional datasets

This folder documents how to add and run additional datasets **locally** without publishing any row-level data.

- `florida_icpsr_27781.md` — Florida prison releases (ICPSR 27781) integration notes (rearrest/reconviction).
- `ncrp_icpsr_37973.md` — NCRP selected variables (ICPSR 37973) integration notes (reincarceration).
